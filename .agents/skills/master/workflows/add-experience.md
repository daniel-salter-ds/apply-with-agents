# Workflow: Add experience

## 1. Interview

Collect: organisation, title, dates, location, role summary, bullet achievements, metrics.

Cross-check against `config/target-role-research.md` for high-yield metrics for the user's target roles.

## 2. Update master.md

Append or insert under `## Experience` in correct reverse-chronological order.

## 3. Record

Note addition in any active role's `notes.md` under `## Master updates` if a role folder is in scope.

## 4. Offer

Ask if user wants to refresh any active tailored `resume.md` via `/resume`.
