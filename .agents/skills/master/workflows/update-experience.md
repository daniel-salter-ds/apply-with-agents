# Workflow: Update experience

## 1. Locate entry

Find employer/role in `master.md`. Confirm with user if ambiguous.

## 2. Interview

What changed? Metrics, scope, dates, title, bullets to add/remove.

## 3. Edit master.md

Apply changes. Do not propagate to role folders automatically — user may run `/resume` per application.

## 4. Record

`## Master updates` in active role `notes.md` if applicable.
